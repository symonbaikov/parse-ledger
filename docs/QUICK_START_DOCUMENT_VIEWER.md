# Quick Start: Transaction Document Viewer

## Быстрый старт за 3 шага

### 1. Откройте хранилище

Перейдите на страницу хранилища:

```
http://localhost:3000/storage
```

### 2. Найдите выписку

Найдите нужную банковскую выписку в списке файлов.

### 3. Нажмите на глаз 👁️

Кликните на иконку глаза рядом с файлом — откроется красиво оформленный документ!

---

## Что вы увидите

### 📋 Красивый документ с:

- **Фиолетовая шапка** с информацией о банке и счете
- **4 карточки** с балансами и итогами
- **Таблица транзакций** со всеми деталями
- **Цветовые индикаторы**: 🟢 зеленый = поступление, 🔴 красный = списание

### 🔧 Доступные действия:

| Кнопка | Действие |
|--------|----------|
| **← Назад** | Вернуться назад |
| **✏️ Редактировать** | Изменить транзакции |
| **🖨️ Печать** | Распечатать документ |

---

## Печать документа

### Быстрая печать:

1. **Клавиатура**: `Ctrl + P` (Windows) или `Cmd + P` (Mac)
2. **Кнопка**: Нажмите иконку принтера в верхней панели

### Сохранение в PDF:

1. Откройте диалог печати
2. Выберите "Сохранить как PDF"
3. Укажите имя файла
4. Нажмите "Сохранить"

### 💡 Совет для лучшего качества:

В настройках печати включите опцию **"Фоновая графика"** или **"Background graphics"** — так сохранятся все цвета и градиенты!

## Примеры использования

### Сценарий 1: Проверка выписки
```
Storage → 👁️ → Просмотр → Проверка данных → ✏️ Редактировать (если нужно)
```

### Сценарий 2: Печать для бухгалтерии
```
Storage → 👁️ → 🖨️ Печать → Сохранить как PDF → Отправить бухгалтеру
```

### Сценарий 3: Презентация
```
Storage → 👁️ → Показать на проекторе
```

## Часто задаваемые вопросы

### ❓ Можно ли редактировать в этом режиме?

**Нет.** Это режим только для просмотра. Для редактирования нажмите кнопку "✏️ Редактировать".

### ❓ Где старая страница редактирования?

**Доступна по клику на "Редактировать"** или напрямую через URL: `/statements/{id}/edit`

### ❓ Почему цвета не печатаются?

**Включите фоновую графику** в настройках печати браузера.

### ❓ Как вернуться в хранилище?

**Нажмите "← Назад"** или используйте кнопку браузера.
